# Student Course Management System

## Project Description
This is a Java-based Student Course Management System to manage students, courses, and enrollments.

## Features
- Add students
- Add courses
- Enroll students into courses
- Handle invalid inputs using custom exceptions

## Technologies Used
- Java
- OOP
- Exception Handling

## Project Structure
- `entity` → model classes
- `service` → business logic
- `exception` → custom exceptions
- `util` → helper classes
- `ui` → main class
